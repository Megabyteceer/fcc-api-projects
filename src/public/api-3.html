API Basejump: URL Shortener


Example creation usage:

https://fcc-basejump-api.herokuapp.com/api-3/new/https://www.google.com
https://fcc-basejump-api.herokuapp.com/api-3/new/http://freecodecamp.com/news

Example creation output:

{ "original_url": "http://freecodecamp.com/news", "short_url": "https://fcc-basejump-api.herokuapp.com/api-3/4" }

Usage:

https://fcc-basejump-api.herokuapp.com/api-3/4

Will redirect to:

http://freecodecamp.com/news
